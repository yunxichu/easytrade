# 外盘商品走势分析与预测系统项目报告

## 项目概述

本项目是一个完整的前后端Web应用系统，用于展示外盘商品（如原油、黄金等）的价格走势，进行期限结构分析，并提供交易策略回测功能。系统采用Flask框架构建后端API，使用原生HTML/CSS/JavaScript构建前端界面，实现了数据获取、可视化展示、智能分析和策略回测等核心功能。

---

## 一、项目目标与任务

### 1.1 主要任务

#### 任务一：外盘商品走势展示与预测
- 获取外盘商品数据（原油、黄金等）
- 展示主力合约最新价、增减仓、合约乘数、期限结构
- 进行价格预测

#### 任务二：期限结构分析
- 获取某一类合约不同月份收盘价
- 绘制期限结构图
- 判断升水或贴水结构
- 提供交易策略建议

#### 任务三：交易策略回测系统
- 构建均线或突破策略
- 实现完整的交易系统（开仓、平仓、资金管理、风控）
- 计算量化指标（胜率、赔率、最大回撤、收益率、夏普比等）

### 3.3 期限结构分析模块

#### 功能特性
1. **获取不同月份合约价格**
   - 近月、次近月、第3月、第4月、第5月、第6月
   
2. **判断升贴水结构**
   - 升水 (Contango): 远月价格 > 近月价格
   - 贴水 (Backwardation): 远月价格 < 近月价格

3. **交易策略建议**
   - **贴水结构**:
     - 多头展期策略：持有多头头寸并展期获得收益
     - 跨期套利：买入近月，卖出远月
   
   - **升水结构**:
     - 空头展期策略：持有空头头寸并展期获得收益
     - 跨期套利：卖出近月，买入远月

#### API接口
```
GET /api/term-structure/<commodity_key>
```

### 3.4 交易策略回测模块

#### 策略设计

##### 双均线策略
- **买入信号**: MA5上穿MA20（金叉）
- **卖出信号**: MA5下穿MA20（死叉）

##### 交易系统
1. **开仓规则**
   - 信号触发：MA5上穿MA20
   - 仓位管理：每次使用10%的资金
   - 执行价格：当日收盘价

2. **平仓规则**
   - 信号平仓：MA5下穿MA20
   - 止损平仓：亏损达到5%
   - 止盈平仓：盈利达到10%

3. **资金管理**
   - 初始资金：$100,000
   - 单笔仓位：总资金的10%
   - 风险控制：单笔最大亏损5%

4. **风险控制**
   - 止损比例：5%
   - 止盈比例：10%
   - 最大回撤监控

#### 量化指标计算

##### 1. 胜率 (Win Rate)
```python
win_rate = winning_trades / total_trades * 100
```

##### 2. 赔率 (Odds Ratio)
```python
odds_ratio = avg_win / avg_loss
```

##### 3. 最大回撤 (Maximum Drawdown)
```python
max_drawdown = max((peak_capital - capital) / peak_capital)
```

##### 4. 总收益率 (Total Return)
```python
total_return = (final_capital - initial_capital) / initial_capital * 100
```

##### 5. 夏普比率 (Sharpe Ratio)
```python
sharpe_ratio = (avg_return - risk_free_rate) / std_return * sqrt(252)
```

#### API接口
```
GET /api/strategy/backtest/<commodity_key>/<period>
```
