## 一、截面多空策略构建详解

### 1.1 核心思想

**截面多空策略（Cross-Sectional Long-Short Strategy）** 的本质，是在每个调仓时点对所有资产按某种因子进行**横截面排名**，买入得分最高的一组（做多腿），卖出得分最低的一组（做空腿），通过组合的多空对冲获取**相对收益（Alpha）**，而非绝对方向性暴露。

与时序策略不同，截面策略的盈利来源不依赖于市场整体方向，而来自于**品种间的相对强弱关系**。这使得它在牛市、熊市中均可运作，是对冲基金中最经典的策略范式之一。

```
时间轴示意：
t₀ ──▶ 按因子排名 ──▶ 构建多空组合 ──▶ t₁ ──▶ 再排名 ──▶ 再调仓 ──▶ t₂
                         ↑ 做多 Top 20%          ↑ 做空 Bottom 20%
```

### 1.2 完整构建流程（七步法）

#### Step 1：标的宇宙（Investment Universe）

标的宇宙的选取是策略的地基，直接影响策略容量和稳健性：

| 筛选维度 | 具体规则 | 目的 |
|---------|---------|------|
| **流动性** | 剔除日均成交量低于阈值的品种 | 避免冲击成本过大 |
| **价格下限** | 剔除价格极低的品种（penny assets） | 避免价格失真 |
| **存活偏差** | 使用**当时点**存在的品种，不能回测时使用现在成分股 | 避免幸存者偏差 |
| **合约流动性** | 外盘期货关注主力合约换月规律 | 避免流动性断层 |

本项目宇宙：12个外盘商品品种（黄金 GC、原油 CL、白银 SI、铜 HG、天然气 NG、铝、锌、镍、大豆、玉米、小麦、棉花），均为全球流动性较好的主要商品合约。

#### Step 2：因子计算

**因子（Factor）** 是用来预测品种相对强弱的定量信号。根据经济学直觉，本项目使用五类因子：

```python
# 1. 短期动量（趋势延续）
MOM_1M = price.pct_change(21)          # 过去21个交易日收益率

# 2. 中期动量
MOM_3M = price.pct_change(63)          # 过去63个交易日收益率

# 3. 波动率倒数（低波动溢价）
VOL_INV = 1 / price.pct_change().rolling(20).std()

# 4. RSI均值回归（超卖品种反弹预期）
delta = price.diff()
gain  = delta.clip(lower=0).rolling(14).mean()
loss  = (-delta.clip(upper=0)).rolling(14).mean()
RSI   = 100 - 100 / (1 + gain / loss)
RSI_REV = 100 - RSI      # 低RSI → 高得分（均值回归方向）

# 5. 均线偏离（趋势跟踪）
MA_DEV = price.rolling(5).mean() / price.rolling(20).mean() - 1
```

#### Step 3：截面 Z-score 标准化（⭐最关键步骤）

因子标准化必须在**每个时间截面上跨品种**进行，而不能在时序上进行。这是截面策略最容易出现的错误之一。

**错误做法**（时序标准化，引入未来泄漏）：
```python
# ❌ 错误：用历史均值/方差标准化，包含未来信息
z = (factor - factor.rolling(252).mean()) / factor.rolling(252).std()
```

**正确做法**（截面标准化，每行是一个时间点）：
```python
# ✅ 正确：每个交易日，对当天所有品种的因子值做标准化
def zscore_cross_section(factor_df):
    """
    factor_df: shape = (T, N)，T=时间步，N=品种数
    每行（每个交易日）独立标准化
    """
    mean = factor_df.mean(axis=1)          # 当日所有品种均值
    std  = factor_df.std(axis=1)           # 当日所有品种标准差
    return factor_df.sub(mean, axis=0).div(std.replace(0, np.nan), axis=0)
```

同时，需要对因子进行**去极值处理（Winsorize）**，避免极端值扭曲截面排名：

```python
def winsorize_mad(x, n_mad=3):
    """3倍MAD去极值（比3sigma更稳健）"""
    median = x.median()
    mad    = (x - median).abs().median()
    upper  = median + n_mad * mad * 1.4826   # 1.4826是正态分布系数
    lower  = median - n_mad * mad * 1.4826
    return x.clip(lower=lower, upper=upper)
```

#### Step 4：多因子合成

将各因子的截面 Z-score 加权合成为综合得分：

| 合成方法 | 公式 | 适用场景 |
|---------|------|---------|
| **等权平均** | `score = Σ z_i / N` | 数据量不足、简单基准 |
| **IC 加权** | `score = Σ IC_i × z_i` | 有足够历史 IC 估计时（推荐） |
| **ICIR 加权** | `score = Σ (IC/σ_IC)_i × z_i` | 更注重 IC 稳定性 |
| **PCA 合成** | 取第一主成分 | 因子高度相关时降维 |

本项目权重设置（等权近似 + 经验调整）：

```python
factor_weights = {
    'MOM_1M':  0.30,   # 短期动量（主要信号源）
    'MOM_3M':  0.30,   # 中期动量（趋势确认）
    'VOL_INV': 0.20,   # 低波动溢价（风险调整）
    'RSI_REV': 0.10,   # 均值回归（反转补充）
    'MA_DEV':  0.10,   # 均线偏离（趋势跟踪）
}
```

#### Step 5：分位数建仓

```python
def generate_signals(factor_score, long_q=0.75, short_q=0.25):
    """
    截面多空分组：
      Top 25%  → +1（做多）
      Bottom 25% → -1（做空）
      中间 50%   →  0（空仓）
    """
    signals = pd.DataFrame(0.0, index=factor_score.index,
                           columns=factor_score.columns)
    for date, row in factor_score.iterrows():
        valid = row.dropna()
        hi = valid.quantile(long_q)   # 75分位
        lo = valid.quantile(short_q)  # 25分位
        signals.loc[date, valid >= hi] =  1.0
        signals.loc[date, valid <= lo] = -1.0
    return signals
```

**等权权重构建**：

```python
def compute_weights(signals):
    weights = signals.copy().astype(float)
    for date, row in signals.iterrows():
        n_long  = (row ==  1.0).sum()
        n_short = (row == -1.0).sum()
        if n_long  > 0: weights.loc[date, row ==  1.0] =  1.0 / n_long
        if n_short > 0: weights.loc[date, row == -1.0] = -1.0 / n_short
    return weights
```

#### Step 6：换仓与成本控制

换仓频率是策略容量与交易成本之间的权衡：

```python
# 交易成本模型
def apply_cost(port_rets, weights, commission=0.0003, slippage=0.0002):
    """
    commission：手续费（双边 3bp，期货约 0.5bp，可调整）
    slippage：  市场冲击（流动性越差越高）
    """
    turnover = weights.diff().abs().sum(axis=1)   # 换手率
    cost = turnover * (commission + slippage)
    return port_rets - cost
```

| 参数 | 典型值（外盘期货） | 说明 |
|------|-----------------|------|
| 手续费 | 0.5~2 bp/单边 | 主流经纪商 |
| 滑点 | 2~5 bp | 取决于合约流动性 |
| 换手率 | 日换仓约 20~40%/天 | 周换仓可降至 5~10%/天 |

#### Step 7：绩效归因指标体系

| 指标 | 公式 | 优秀标准 |
|------|------|---------|
| **IC（信息系数）** | `corr(factor_t, return_{t+1})` | > 0.05 |
| **ICIR** | `mean(IC) / std(IC)` | > 0.5 |
| **年化收益率** | `(1+R)^252/T - 1` | > 10% |
| **夏普比率** | `E[R-Rf] / σ × √252` | > 1.0 |
| **最大回撤** | `max(peak-trough)/peak` | < 15% |
| **Calmar 比率** | `年化收益 / |最大回撤|` | > 1.0 |
| **胜率** | `P(日收益>0)` | > 52% |

## 二、动量策略文献研究综述

### 2.1 动量效应的发现与奠基

#### 2.1.1 Jegadeesh & Titman（1993）——动量效应的奠基之作

> *"Returns to Buying Winners and Selling Losers: Implications for Stock Market Efficiency"*  
> **Journal of Finance**, Vol. 48, No. 1, pp. 65–91.

这篇论文是动量策略研究的里程碑，研究发现：

- 在1965—1989年美国股票市场中，**过去3~12个月的「赢家」组合，未来3~12个月仍能显著跑赢「输家」组合**
- 著名的 **(J, K) 策略**：形成期 J 个月，持有期 K 个月
- 经典参数 (12, 1)（即过去12个月动量，持有1个月）年化超额收益约 **12%**
- 该效应无法被 CAPM 的系统性风险（beta）所解释

**关键发现**：动量收益在持有期 3~12 个月最强，超过 12 个月后逐渐反转（均值回归）。

#### 2.1.2 Carhart（1997）——动量成为第四因子

> *"On Persistence in Mutual Fund Performance"*  
> **Journal of Finance**, Vol. 52, No. 1, pp. 57–82.

在 Fama-French 三因子（市场、规模、价值）基础上加入**动量因子（MOM）**，构建四因子模型：

```
E[Ri] - Rf = αi + βi·(Rm-Rf) + si·SMB + hi·HML + mi·MOM + εi
```

其中 MOM = 过去12个月（**跳过最近1个月**）的等权长短组合收益差。

跳过最近1个月的原因：最近1个月内存在**微观市场摩擦导致的短期反转效应**，若不跳过会污染中期动量信号。

#### 2.1.3 Fama & French（2012）——全球市场验证

> *"Size, Value, and Momentum in International Stock Returns"*  
> **Journal of Financial Economics**, Vol. 105, No. 3, pp. 457–472.

在北美、欧洲、亚太四个市场均发现显著动量效应，但有一个著名例外：

> **日本市场的动量效应极弱，至今是学术界的未解之谜。**

可能原因：日本上市公司与银行的交叉持股结构、企业文化导致的盈利公告延迟、散户行为模式的差异。

### 2.2 动量效应的理论解释

#### 2.2.1 行为金融学解释（主流）

| 理论 | 提出者 | 核心机制 |
|------|--------|---------|
| **信息扩散不足** | Hong & Stein (1999) | 信息在市场中缓慢扩散（「新闻观察者」vs「动量交易者」），价格对好消息反应缓慢，导致持续上涨 |
| **过度自信** | Daniel, Hirshleifer & Subrahmanyam (1998) | 投资者对私有信息过度自信，对公开信息反应不足；初期推动价格，后期随信息曝光反转 |
| **保守性偏差** | Barberis, Shleifer & Vishny (1998) | 投资者低估公司盈利的持续性（锚定于均值），好消息来临时股价上涨不足，后续缓慢追涨 |
| **处置效应** | Grinblatt & Han (2005) | 投资者倾向于过早卖出盈利股（锁定收益）、惜售亏损股（不愿确认损失），导致赢家继续上涨、输家继续下跌 |
| **注意力有限** | Peng & Xiong (2006) | 有限注意力导致投资者只关注热门股，信息传播不均匀 |

#### 2.2.2 风险因子解释（少数）

- Johnson (2002)：动量与经济增长率的期权性质有关
- Ahn et al. (2003)：条件 CAPM 中时变 beta 可部分解释动量
- **但大多数研究表明：动量是真正的定价异常，不能被标准风险因子完全解释。**

### 2.3 商品市场的动量——与本项目直接相关

#### 2.3.1 时序动量（Trend-Following）

> **Moskowitz, Ooi & Pedersen (2012)**  
> *"Time Series Momentum"*  
> **Journal of Financial Economics**, Vol. 104, No. 2, pp. 228–250.

这是商品量化领域最重要的论文之一：

- 研究了 **58 个品种**（股票期货、债券期货、商品期货、外汇），时间跨度 1985—2009
- **时序动量（TSMOM）**：若品种在过去12个月收益为正，则做多；为负则做空
- **所有58个品种均呈现显著的12个月时序动量**
- 组合夏普比率约 **1.0**，与股票、债券、商品等传统资产类别相关性极低（约 0）
- 动量在**2008年金融危机**中表现优异（趋势跟踪的最大优势）

> 「趋势是商品期货市场最持久、最稳健的异常。」—— Moskowitz et al.

**时序动量 vs 截面动量的区别**：

| 维度 | 时序动量（TSMOM） | 截面动量（CSMOM） |
|------|-----------------|-----------------|
| 基准 | 以0收益为基准（绝对动量） | 以品种间排名为基准（相对动量） |
| 市场中性 | 不完全中性（受市场趋势影响） | 多空对冲，市场中性 |
| 代表策略 | CTA/趋势跟踪基金 | 多空股票对冲基金 |
| 商品表现 | 更强（价格延续趋势） | 也显著（约7-9%年化） |

#### 2.3.2 跨资产动量

> **Asness, Moskowitz & Pedersen (2013)**  
> *"Value and Momentum Everywhere"*  
> **Journal of Finance**, Vol. 68, No. 3, pp. 929–985.

在 8 个资产类别（美国股票、英国股票、欧洲股票、日本股票、股票指数期货、债券期货、商品期货、外汇）中同时验证：

- **截面动量在商品期货中年化超额收益约 7~9%**
- 动量效应跨资产类别普遍存在，且各市场内的动量信号具有一定相关性（「全球动量」）
- 价值因子与动量因子在同一资产类别内**负相关**，组合后效果大幅改善

### 2.4 动量因子演化谱系

```
动量因子演化树（1993 → 今）
│
├── 原始动量（Jegadeesh & Titman 1993）
│   ├── 截面动量：跨品种排名做多/做空
│   └── 时序动量：单品种与0对比（Moskowitz 2012）
│
├── 残差动量（Residual Momentum）
│   └── Blitz, Huij & Martens (2011)：剔除三因子后的特质动量
│       → IC更高，受宏观因子干扰更少
│
├── 52周高点动量（George & Hwang 2004）
│   └── 相对于52周最高价的比率，比传统动量更稳健
│       → 锚定心理解释
│
├── 行业动量（Moskowitz & Grinblatt 1999）
│   └── 行业层面动量可解释大部分个股动量
│
├── 双动量（Antonacci 2014）
│   ├── 绝对动量（时序）：熊市/负动量时退出市场
│   └── 相对动量（截面）：选择最强品种
│       → 通过绝对动量过滤规避动量奔溃
│
├── 盈利动量（Earnings Momentum, PEAD）
│   └── Bernard & Thomas (1989)：盈利公告后价格漂移（SUE效应）
│
└── Frog-in-Pan 动量（Da, Gurun & Warachka 2014）
    └── 持续小涨比偶尔暴涨的预测力更强
        → 一致性加权动量
```

## 三、动量奔溃：成因、应对与新因子

### 3.1 动量奔溃的定义与规律

> **Daniel & Moskowitz (2016)**  
> *"Momentum Crashes"*  
> **Journal of Financial Economics**, Vol. 122, No. 2, pp. 221–247.

动量奔溃（Momentum Crash）是指动量策略在特定市场环境下出现**灾难性亏损**的现象——这些亏损往往发生在经济最脆弱的时刻，具有极强的尾部风险特征。

#### 历史典型奔溃事件

| 时间 | 背景 | 动量策略单月亏损 |
|------|------|---------------|
| **1932年7—8月** | 大萧条后期市场反弹 | **-91.59%** |
| **2001年1月** | 科技股泡沫后期反弹 | -45.3% |
| **2009年3—5月** | 金融危机后政府救市反弹 | **-73.42%** |

这些亏损数字触目惊心，远超普通的止损能力范围。

### 3.2 动量奔溃的机制解析

动量奔溃发生需满足三个条件的叠加：

```
条件1：市场在过去24个月大幅下跌（熊市形成期）
         ↓
条件2：市场在过去1个月快速反弹（救市/情绪逆转）
         ↓
条件3：「输家组合」隐含波动率极高（做空腿被逼空）
         ↓
结果：  动量策略双边亏损（多腿滞涨 + 空腿被逼空）
```

**机制深挖**：

- **熊市中的动量组合结构**：
  - 「做多腿」= 防御性资产（消费必需品、黄金、债券），在熊市中保值
  - 「做空腿」= 超跌的周期性资产（金融股、能源股），在熊市中被过度抛售

- **市场反弹时的发生**：
  - 超跌资产（空腿）急速反弹 → 做空头寸承受巨额浮亏 → 被迫止损 → 进一步推涨 → 螺旋逼空
  - 防御性资产（多腿）相对滞涨 → 多头贡献微弱
  - **净效果：双边亏损叠加，策略净值暴跌**

- **期权化解释（Daniel & Moskowitz）**：
  - 动量策略在熊市中相当于**隐性空头头寸上的深度虚值看跌期权**
  - 正常市场中这个期权几乎没有价值（策略表现良好）
  - 市场反转时期权突然大幅行权（策略崩溃）

### 3.3 应对动量奔溃的四大方法

#### 方法一：波动率缩放（Vol-Scaling）⭐推荐

> **Barroso & Santa-Clara (2015)**  
> *"Momentum Has Its Moments"*  
> **Journal of Financial Economics**, Vol. 116, No. 1, pp. 111–120.

这是目前实证效果最好的动量奔溃应对方案：

```python
def vol_scaled_momentum(mom_returns, target_vol=0.12, lookback=126):
    """
    将动量策略标准化至恒定目标波动率
    
    原理：动量奔溃前，策略波动率会显著升高（预警信号）
    通过缩减高波动期的仓位，自然降低奔溃期损失
    
    参数：
        target_vol:  目标年化波动率（12%是常见设定）
        lookback:    估计历史波动率的回望窗口（126天≈半年）
    """
    realized_vol = mom_returns.rolling(lookback).std() * np.sqrt(252)
    scaling_factor = target_vol / realized_vol.shift(1)  # 使用昨日波动率，避免未来泄漏
    scaling_factor = scaling_factor.clip(0.1, 3.0)       # 限制杠杆范围在0.1x~3x
    return mom_returns * scaling_factor
```

**效果**：
- 最大回撤从 -73% 降至约 -20%
- 夏普比率从 ~0.5 提升至 ~1.0
- **年化收益几乎不下降**（关键优势）

#### 方法二：绝对动量过滤（Dual Momentum）

> **Antonacci (2014)**，*Dual Momentum Investing*

```python
def dual_momentum_filter(asset_signal, market_return_12m, rf_rate=0.01):
    """
    双动量：截面动量（相对）+ 时序动量（绝对）结合
    
    规则：
    - 先用截面动量选出最强品种
    - 再用绝对动量检验：若市场过去12月收益 < 无风险利率，则全部退出市场
    
    这一过滤极大减少了在系统性熊市中的暴露
    """
    if market_return_12m < rf_rate:
        return 0.0   # 牛熊过滤：熊市信号，全仓空仓
    return asset_signal
```

**商品期货应用**：用商品综合指数的12个月收益作为市场信号，当整体商品市场趋势为负时，停止新开多仓。

#### 方法三：动态仓位（时变风险暴露）

> **Daniel & Moskowitz (2016)** 原文方法

```python
def dynamic_momentum_exposure(mom_signal, pred_variance, target_var=0.04):
    """
    根据预测方差动态调整动量组合暴露
    
    核心：动量奔溃前，组合方差显著升高（预测方差是先行指标）
    通过反比于方差调整仓位，在高风险期自动降仓
    
    参数：
        pred_variance: 由GARCH或滚动方差预测的下期方差
        target_var:    目标年化方差（4% = 年化波动率20%）
    """
    raw_weight = target_var / pred_variance
    # 熊市额外降仓：当市场过去2年为负时减半暴露
    bear_market = is_bear_market()   # 市场过去24个月收益 < 0
    if bear_market:
        raw_weight *= 0.5
    return raw_weight * mom_signal
```

#### 方法四：跳过最近1个月（Skip-Month）

最简单但有效的方法，已被纳入标准动量因子定义：

```python
# 标准12-1动量（跳过最近21个交易日）
mom_12_1 = price.shift(21) / price.shift(252) - 1

# 原因：最近1个月内，短期反转效应（bid-ask bounce, 流动性压力）
# 会污染中期动量信号，跳过后IC显著提升
```

### 3.4 动量奔溃催生的新因子

动量奔溃的研究不仅是风险管理问题，更催生了一批**更稳健的新一代动量因子**：

#### 新因子一：52周高点动量（52-Week High Momentum）

> **George & Hwang (2004)**  
> *"The 52-Week High and Momentum Investing"*  
> **Journal of Finance**, Vol. 59, No. 5.

```python
def factor_52w_high(price, lookback=252):
    """
    相对52周最高价的距离，比传统动量更稳定
    
    心理学基础：投资者以52周高点为参考锚点（Anchoring Bias）
    当价格接近52周高点时，投资者犹豫不决（不愿意突破历史高点）
    实际突破后往往有更强的趋势延续
    """
    high_52w = price.rolling(lookback).max()
    return price / high_52w   # 越接近1，信号越强（做多）
```

**优势**：
- 比传统动量在极端市场中更稳定
- 不容易出现极端的动量奔溃
- George & Hwang 的研究显示其 IC 优于传统 12-1 动量

#### 新因子二：残差动量（Residual Momentum）

> **Blitz, Huij & Martens (2011)**  
> *"Residual Momentum"*  
> **Journal of Empirical Finance**, Vol. 18, No. 3.

```python
from sklearn.linear_model import LinearRegression

def residual_momentum(asset_returns, factor_returns, lookback=252, signal_window=21):
    """
    用Fama-French三因子（或五因子）回归后的**残差**计算动量
    
    原理：剔除了市场beta、规模、价值因子对收益的解释，
         残差代表纯粹的特质性动量（Alpha动量）
    
    优势：
    1. IC比传统动量提高约20~30%
    2. 动量奔溃幅度减小约50%（因为奔溃主要来源于系统性因子）
    3. 与价值因子的负相关减弱，组合更稳定
    """
    reg = LinearRegression()
    reg.fit(factor_returns[-lookback:], asset_returns[-lookback:])
    residuals = asset_returns[-lookback:] - reg.predict(factor_returns[-lookback:])
    return residuals[-signal_window:].mean()   # 最近1个月残差均值为信号
```

#### 新因子三：Frog-in-Pan 动量（稳健动量）

> **Da, Gurun & Warachka (2014)**  
> *"Frog in the Pan: Continuous Information and Momentum"*  
> **Review of Financial Studies**, Vol. 27, No. 7.

名字来自温水煮青蛙的寓言：
- **持续小幅上涨**（不引人注意）→ 投资者反应不足更严重 → 未来动量更强
- **一次性暴涨**（引人注目）→ 吸引大量关注 → 反应更充分 → 动量更弱

```python
def frog_in_pan_momentum(daily_returns, window=252):
    """
    Frog-in-Pan动量：考虑收益方向一致性
    
    FIP = (总收益) × (正收益天数占比)
    一致性越高，未来动量越强
    """
    total_ret = daily_returns[-window:].sum()
    consistency = (daily_returns[-window:] > 0).mean()   # 正收益天数比例
    return total_ret * consistency
```

#### 新因子四：条件动量（Conditional Momentum）

```python
def conditional_momentum(mom_score, market_state):
    """
    熊市/牛市条件下动量暴露不同
    
    基于Daniel & Moskowitz (2016)的发现：
    - 牛市中：全力动量（动量溢价正常）
    - 熊市中：大幅降低动量暴露，甚至反转（逆动量）
    
    market_state: +1=牛市，-1=熊市（基于过去12/24个月市场收益判断）
    """
    if market_state == 1:    # 牛市
        return 1.0 * mom_score
    elif market_state == -1: # 熊市
        return -0.2 * mom_score   # 轻微逆动量
    else:
        return 0.5 * mom_score    # 中性
```

#### 新因子五：机器学习动量增强

> **Gu, Kelly & Xiu (2020)**  
> *"Empirical Asset Pricing via Machine Learning"*  
> **Review of Financial Studies**, Vol. 33, No. 5.

使用 LSTM、梯度提升等机器学习模型非线性组合动量信号：

```python
# 概念示意（非完整实现）
class MLMomentum:
    """
    用机器学习非线性组合多个动量信号
    
    输入特征：
    - 12-1 动量
    - 52周高点比率
    - RSI
    - 波动率
    - 成交量趋势
    - 宏观环境变量（VIX、利差等）
    
    输出：综合多空信号（比线性加权IC提高约15~25%）

```
